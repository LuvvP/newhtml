HTML/CSS đều là hai ngôn ngữ lập trình. Trong khi HTML (HyperText Markup Language – ngôn ngữ đánh dấu) được dùng để xây dựng cấu trúc cho từng phần nội dung thì CSS (Cascading Style Sheet- ngôn ngữ định dạng theo từng lớp) được dùng để tạo định dạng hiển thị cho HTML.

Giới thiệu về HTML

HTML

- HTML – HyperText Markup Language (Ngôn ngữ đánh dấu siêu văn bản) : giúp người dùng tạo và cấu trúc các thành phần trong trang web hoặc ứng dụng, phân chia các đoạn văn ,heading,links,…

Giới Thiệu về CSS

CSS là gì

CSS (Casading Style Sheets) được hiểu 1 cách đơn giản đó là cchs mà chúng ta thêm các kiểu hiện thị (font chữ,kích thước,màu nền, ảnh,….) cho một trang Web HTML

CSS nâng cao

  • ID,Class,Selector
  • Attribute nâng cao
  • Pseudo – class/element
  • Flexbox
  • Animations
